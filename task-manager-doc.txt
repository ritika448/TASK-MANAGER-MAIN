Project Overview ­ Task Manager App

27 June 2025 01:20 PM

Project Overview:

A multi-user task management web application designed to enable users to create, assign, track, and
manage tasks efficiently. Offers card and list views, calendar scheduling, sorting, grouping, recurring
tasks.

Objective:

- Simple yet powerful task manager for teams.
- Access for task creation, assignment, and calendar view.
- Recurring task support for routine workflows.
- Visual dashboards and responsive UI for a modern experience.

Technical Requirements:

   · Frontend: React.js (Vite), Tailwind CSS, MUI, Typescript.
   · Backend: .Net 8.0,Repository Pattern, Dapper
   · Database: SQL Server
   · Authentication: JWT Authentication
   · File Storage: Azure Blob Storage
   · Other Services: Azure Communication Service- Email
Backend Project structure
Solution name: TaskManager.sln
    TaskManager.API
    TaskManager.DAL
    TaskManager.Model
    TaskManager.Database

Modules:

   · Login Management

        - Para: EmailID
        - Check for user's existence.

       2. /ValidateResetPasswordCode
        - Para: ResetPasswordCode

        - Else return Ok()

        - Request: ResetPasswordCode, Password
        - Check whether the code is valid or not.
        - If code does not exists, return "Invalid Code."
        - If code exists but is expired, return "Code is expired."
        - If code is Valid, then update the password.
